# VUE.JS

### 1.VUE CLI 설치

- node 설치 https://nodejs.org/en/download/
- https://cli.vuejs.org/
- vue cli 설치 확인 vue --version

### 2. VUE Project 설치

- To create a new project, run:

```cmd
	vue create hello-world
```

- To create a new project with ui, run:

```cmd
	vue ui
```

- install 및 실행방법

```cmd
 	npm install
	npm run serve
```
