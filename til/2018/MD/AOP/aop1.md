##AOP란?
* Aspect로 흩어진 관심사(비슷한 코드 EX 로그, 트랜젝션 등등) 를 모아서 모듈화할수있다.

* AOP의 주요개념
   1. Aspect : 흩어진 관심사의 묶음 : 모듈
   2. Advise  : 모듈이 해야할 일들
   3. pointcut :모듈이 어디에 적용되어야하는지 
   4. target : ADVISE 적용 대상
   5. join point : 메서드 실행시점 합류지점
   
 * AOP 구현체
    1. AspectJ
    2. 스프링AOP
      
 * AOP 적용방법
    1. 컴파일
    2. 로드타임
    3. 런타임    : 스프링 AOP 에서 사용하는 방법으로 최초의 빈 생성 시 적용되며 별도의 설정등을 할 필요가 없음
